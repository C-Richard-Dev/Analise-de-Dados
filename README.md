Este repositório contém um projeto de análise de dados utilizando Jupyter Notebook. 
O arquivo graficos.ipynb apresenta visualizações gráficas criadas com Plotly, enquanto cancelamentos_sample.csv contém dados de amostra para análise de cancelamentos.
É um projeto que acompanha desde o tratamento das informações da base de dados até as insights valiosas extraidas da analise dos graficos.
